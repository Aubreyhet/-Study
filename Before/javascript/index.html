UI事件

  • DOMActivate

    已经废弃，不建议使用

  • load

    当页面完全加载后在window上触发

  • unload

    当页面完全卸载后在window上触发

  • abort

    在用户停止下载过程时，如果嵌入的内容没有加载完，就在object 上触发

  • error

    当javascript发生错误时在window上触发

  • elect

    当用户选择文本框中一个或者多个字符时在该元素上触发

  • resize

    当窗口或者框架的大小发生变化时，在window或者框架上触发

  • scoll

    当用户滚动带滚动条的内容时在该元素上触发

  • load事件

    两种定义方式

    1.使用指定事件处理程序的方式添加lode事件

    2.使用在body上添加onlode事件

  • unlode事件

    • unlode事件与lode事件相似 定义也是有两种相同的方式。

    lode和unlode事件也可以绑定在img和input上，当元素加载完成或者卸载完成时触发事件

  • resize 事件

    这个事件也是可以通过指定事件处理程序的方式添加

  • scoll事件

焦点事件

  • blur事件

    当元素失去焦点之后触发，这个事件不会冒泡

  • DOMFocusIn

    dom3废弃了这个事件可以使用focusin事件

  • DOMFocusOut

    同上，我们可以使用focusout

  • focus

    在元素获得焦点时触发该事件，该事件不冒泡

  • focusin

    focusin 事件 在元素在获得焦点时触发 但是他会冒泡

  • focusout

    focusout 事件在元素失去焦点时触发，相当于blur事件

    鼠标和滚轮事件

    • click

      当用户单机鼠标或者按下回车时触发

    • dblcilck

      当用户双击鼠标时触发该事件

    • mousedown

      当用户按下鼠标任意按键时触发该事件

    • mouseenter

      当用户首次从元素外移动到元素范围内的时候触发该事件，不会冒泡

    • mouseleave

      光标移动离开元素时触发该事件，不会冒泡

    • mousemove

      当光标在元素内移动时重复触发

    • mouseout

      拖出元素时触发

    • mouseover

      拖入元素时触发的事件

    • mouseup

      在用户释放鼠标时触发事件
